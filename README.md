# Life with Instance

A life manager application developed with Flutter. 

## Dependency

- Flutter: https://flutter.io/
- Dart 2: https://www.dartlang.org/

## Usage

```shell
git clone https://github.com/yuwenhuisama/LifeWithInstance.git
cd LifeWithInstance
flutter build apk
```

## Route Map

- Local Storage
- Online Account
- Network Storage
